    for gamma in gamma_values if kernel == 'rbf' else [None]:  # gamma is relevant only for 'rbf'
        
        # Initialize and fit Kernel PCA with current parameters
        if gamma:
            kernel_pca = KernelPCA(n_components=n_components, kernel=kernel, gamma=gamma)

Linear Regression for Predicting S11: Linear regression was straightforward to implement and interpret, providing an initial understanding of the relationships between design parameters and S11 components. In summary, PCA and linear regression models provide a good starting point for simplification but may not fully capture the complexities of antenna performance due to their linear assumptions.

Linear Regression Limitations: Linear regression assumes a linear relationship between features and the target, which is likely too simplistic for modeling complex electromagnetic behavior. 5. Summary¶

Methodologies¶

1. Data Preprocessing:

The design parameters were standardized before applying PCA. Standardization was crucial because PCA is sensitive to the scale of the input data. By centering the data to have a mean of 0 and a standard deviation of 1, we ensured that each feature contributed equally to the principal components.

2. Dimensionality Reduction with PCA:

We applied Principal Component Analysis (PCA) to the standardized design parameters to reduce the dimensionality of the data.

This was necessary because high-dimensional data can be noisy and computationally intensive to model.

We selected the number of components by analyzing the cumulative explained variance plot, targeting components that collectively explain a significant portion of the variance while simplifying the feature set. In this case, using 8 components provided a reasonable balance, retaining most of the relevant variance.

3. Kernel PCA for Nonlinear Patterns:

Given that the relationships between design parameters and S11 values are likely nonlinear, we experimented with Kernel PCA (KPCA), specifically with an RBF kernel.

Kernel PCA aims to capture nonlinear patterns that standard PCA might miss by transforming the data into a higher-dimensional space where linear separability is more achievable.

4. Linear Regression Models:

After dimensionality reduction, linear regression was performed on the transformed principal components to predict the real and imaginary parts of the S11 parameter at key frequency points. Linear regression was chosen for its simplicity and interpretability, serving as a baseline to evaluate how well linear relationships could approximate the data.

5. Model Evaluation:

The model’s performance was assessed using Mean Squared Error (MSE) and ( R^2 ) scores for both the real and imaginary parts of S11. MSE indicates the average prediction error, while ( R^2 ) measures the proportion of variance explained by the model. Low ( R^2 ) values suggest that the linear model may struggle to capture the complexity of the data.

Findings¶
  1. PCA Results

The results from applying PCA followed by linear regression show moderate Mean Squared Error (MSE) scores but relatively low (R^2)values:

Real Part: MSE: 0.1860 (R^2): 0.1488

Imaginary Part: MSE: 0.1303 (R^2): 0.0901

These low (R^2) scores indicate that while the model can predict some aspects of the S11 parameter’s behavior, a significant portion of the variance remains unexplained. This suggests that the relationships between the design parameters and S11 are likely more complex than what a linear model can capture.

  1. Kernel PCA Results

When using Kernel PCA with an RBF kernel (gamma set to 0.01), the results did not show substantial improvements over standard PCA. The MSE and ( R^2 ) values remained similar, indicating that while Kernel PCA introduced some nonlinearity, it may not have fully captured the complexity needed for accurate predictions.

This finding implies that although nonlinearity exists in the data, it may require more sophisticated modeling techniques or further tuning of parameters to effectively capture these relationships.

Patterns Observed:¶

The actual vs. predicted plots showed that the linear regression model, even with dimensionality reduction, struggles with dispersion around the ideal fit line. This dispersion implies that the relationship between the input parameters and S11 is likely nonlinear.

Certain frequency points (especially near resonance points) were more accurately predicted than others, indicating that the model might be picking up on patterns related to specific resonant behaviors but lacks precision across the entire dataset.

Conclusions and Real-World Implications¶

Linear Models’ Limitations:

Linear regression is limited in capturing complex, nonlinear relationships between design parameters and antenna performance metrics like S11. This aligns with real-world observations in antenna design, where electromagnetic behaviors are often highly nonlinear due to interactions between geometry, materials, and frequency.

Dimensionality Reduction is Useful but Not Sufficient:

PCA and Kernel PCA helped simplify the design space, making the regression task more manageable. However, dimensionality reduction alone cannot account for nonlinearity in relationships. Real-world antenna design likely requires nonlinear models to achieve accurate predictions of parameters like S11 across varying frequencies.

Challenges and Solutions¶

1. Challenge: Capturing Nonlinearity

  • Problem: The primary challenge was capturing the complex, nonlinear relationship between design parameters and S11 values. Linear regression inherently assumes a straight-line relationship, which doesn’t fully represent the behaviors in electromagnetic simulations.

  • Solution: We introduced Kernel PCA to capture some nonlinear patterns, but the results showed only marginal improvement. To overcome this challenge more effectively, experimenting with more sophisticated nonlinear regression models (like Support Vector Regression, Gradient Boosting, or Neural Networks) would likely yield better results.

2. Challenge: Dimensionality Reduction Trade-offs

  • Problem: Balancing the number of components in PCA or Kernel PCA was challenging. Reducing dimensions too aggressively risks losing critical information, while keeping too many components makes the model more complex and computationally intensive.

  • Solution: We used explained variance as a guide to select the number of components. However, this approach could be improved by testing more components or using domain knowledge to prioritize specific features tied to S11 behavior.

3. Challenge: Parameter Tuning in Kernel PCA

  • Problem: Selecting optimal parameters (such as gamma for the RBF kernel in Kernel PCA) was challenging, as it required trial and error to find a setting that effectively captured the data structure.

  • Solution: We tested multiple values for gamma and other kernels. To improve this, a more systematic parameter search (e.g., grid search or random search) could be applied to optimize Kernel PCA settings and potentially improve the model’s ability to capture nonlinearity.

Future Directions¶

To improve the model further, the following steps could be beneficial:

Experimenting with Nonlinear Models: Try Support Vector Regression, Random Forest, or Neural Networks, as these models can capture complex interactions and nonlinear patterns that linear regression cannot.

Feature Engineering: Consider creating additional features, such as interaction terms or nonlinear transformations, that could help linear models capture more complex relationships.

Hyperparameter Tuning with Cross-Validation: For Kernel PCA and other models, using grid search or cross-validation to tune parameters could improve model performance and help identify the best settings more efficiently.

In conclusion, while PCA and linear regression provide a useful starting point for understanding antenna performance, the complexity of the data suggests that nonlinear modeling approaches and additional tuning would likely yield better predictive accuracy for real-world applications in antenna design.
